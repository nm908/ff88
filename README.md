# 🎰 FF88 - Gaming Platform

**FF88** là một nền tảng chơi game mô phỏng sử dụng **điểm ảo** (không tiền thật) với giao diện chuyên nghiệp và 100 AI players.

## 🎮 Games Available

- **CRASH GAME** 📈: Dự đoán multiplier cao nhất. Mỗi 0.01x = cộng điểm. Càng cao càng khó, xác suất thắng giảm.
- **ĐUA NGỰA** 🐎: Cược vào con ngựa may mắn. Thắng x2 lãi, thua mất cược.
- **BÓNG ĐÁ** ⚽: Dự đoán tỉ số trận đấu. Thắng x2, thua mất cược.
- **TÀI - XỈU** 🎲: 3 xúc xắc. Tài (11-18) hay Xỉu (3-10). Thắng x2, thua mất cược.
